### 使用方法
```bash
curl -LO https://github.com/teaper/archlinux-install-script/releases/download/0.1.9/archlinux-install.sh
sudo bash archlinux-install.sh
```

- [x] 安装 ArchLinux 操作系统

- [x] 安装桌面环境
